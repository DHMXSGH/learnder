在这篇指南里，主要针对初学者推荐一些视频教程，帮助大家快速入门。

你可以通过给出的关键词决定是否观看。

::: tip 提示

- 部分视频源 Youtube（~~国际版优酷~~），你需要一些特殊方式才能观看。如果真的没有办法，可以通过在 [Bilibili](https://search.bilibili.com/) 搜索原标题看看有没有人搬运。

- 建议进入视频源站进行观看，部分视频是系列制作的，如果你只看其中一集，可能会有些不知所云。

:::
